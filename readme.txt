== Description ==

A simple jamaoni game for wordpress. just place [JAMAONI] in your post and pages.The main concept behind this games is to create any type of figure through blocks.every click on star icon will provide a new block.user can drag & drap that block anywhere on the page. user can change the color of block and can delete unwanted blocks.

== Installation ==

1. Upload `jamaoni` to `/wp-content/plugins/` directory
2. Activate plugin through the 'Plugins' menu in WordPress
3. put [JAMAONI] in your post or pages.
